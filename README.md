Number Guessing Game
Objective
The objective of this project is to create a simple Number Guessing Game using Java. The program generates a random number, and the user has to guess it. The game provides hints whether the guessed number is too high or too low until the correct number is guessed.

Features
Random number generation between 1 and 100.
User input through console.
Hints for incorrect guesses.
Counts the number of attempts.
Displays a success message when the correct number is guessed.
Technologies Used
Java
VS Code / Any Java IDE
Java Scanner Class
Java Random Class
Steps Performed
Generated a random number using the Random class.
Accepted user input using the Scanner class.
Compared the user's guess with the generated number.
Displayed hints ("Too High" or "Too Low").
Continued the game until the correct number was guessed.
Displayed the total number of attempts taken.
How to Run
Save the file as NumberGuessingGame.java.

Open the terminal in the project folder.

Compile the program:

javac NumberGuessingGame.java
Run the program:

java NumberGuessingGame
